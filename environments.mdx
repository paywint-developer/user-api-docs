## Sandbox
1. Contact [support@paywint.com](mailto:support@paywint.com) to request a sandbox (test) account.
2. You’ll receive credentials to access the [Sandbox Dashboard](https://sandbox.app.paywint.com).
3. Navigate to the [Credentials tab](https://sandbox.app.paywint.com/app/developer-panel/api-credentials) to obtain your API key for accessing all sandbox APIs.
4. Set the API base URL to: `https://api.sandbox.paywint.com`

## Production
Before switching to production, please contact our support team to enable API access for your production account. We’re happy to assist with any questions or issues during setup.

1. Contact [support@paywint.com](mailto:support@paywint.com) to request a production account.
2. You’ll receive credentials to access the [Production Dashboard](https://app.paywint.com).
3. Navigate to the [Credentials tab](https://app.paywint.com/app/developer-panel/api-credentials) to obtain your production API key.
4. Set the API base URL to: `https://api.paywint.com`
